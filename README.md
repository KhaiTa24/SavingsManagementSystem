# Bank Savings Account Management System

## Overview
This project is a banking system for managing savings accounts, supporting multiple user roles and transaction workflows.

## Features
- Account opening and early withdrawal  
- Transaction authentication using PIN + OTP  
- Two-step transaction approval  
- State-based transaction flow (request, pending, completed, canceled)  

## User Roles
- Customer  
- Teller  
- Manager  
- Admin  

## Technologies Used
- ASP.NET Core  
- SQL Server  
- UML, BPMN  

## Database Design
- 14 normalized tables  
- Applied constraints and business rules to ensure data integrity  

## Testing
- 13 test cases covering main scenarios and edge cases  

## Notes
This is an academic project developed for system analysis and design practice.
